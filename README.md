# Weather App

A simple Weather Application built using Python and Tkinter.  
This application fetches real-time weather data from a Weather API and displays it in a graphical user interface (GUI).

---

## Features

- Shows current temperature
- Displays weather climate
- Shows weather description
- Displays atmospheric pressure
- Simple GUI using Tkinter
- Fetches live weather data from Weather API

---

## Technologies Used

- Python
- Tkinter
- Requests Library
- Weather API

---

## Run the Project

### Install Required Library

```bash
pip install requests
```

### Run the Application

```bash
python weather_app.py
```
